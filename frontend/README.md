# FoodyX - React + Vite Project

FoodyX is a minimal and fast React project setup using **Vite**, designed for a responsive food ordering platform. It comes with Hot Module Replacement (HMR) and ESLint rules for clean and maintainable code.

## Features

- Fast setup using **Vite** for React
- Hot Module Replacement (HMR) for instant updates during development
- ESLint pre-configured for code consistency
- Responsive design and ready for production deployment
- Support for TypeScript integration if needed

## Official Plugins Available

Currently, two official plugins are available for React projects in Vite:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react)  
  Uses **Babel** for Fast Refresh and JSX transformation.

- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc)  
  Uses **SWC** (a super-fast compiler) for Fast Refresh and JSX transformation.

## Recommended ESLint Setup

If you are developing a **production-ready application**, it is recommended to use **TypeScript** with type-aware lint rules enabled.  

You can check out the [Vite TypeScript template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for more information on integrating **TypeScript** and [`typescript-eslint`](https://typescript-eslint.io) into your project.
